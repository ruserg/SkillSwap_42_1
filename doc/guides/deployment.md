# Развертывание

Документация по деплою приложения.

## Сборка

```bash
npm run build
```

Создает папку `dist/` с готовым к деплою кодом.

## Переменные окружения

Убедитесь, что в продакшене настроены:

```env
VITE_API_BASE_URL=https://api.example.com
```

## Предпросмотр

```bash
npm run preview
```

Проверяет сборку локально перед деплоем.

## Рекомендации

- Минификация включена автоматически
- Оптимизация изображений
- Code splitting через Vite
- Проверка CORS на сервере
